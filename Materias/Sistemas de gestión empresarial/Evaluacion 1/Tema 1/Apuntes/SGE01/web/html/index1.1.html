Descripcion textual alternativa del video modelo cliente/servidor

En este video se explica lo siguiente:

  • Los elementos del modelo cliente/servidor y que son:
    • Servidor: el que recibe la petición.
    • Cliente: el que envíe la petición.
  • Características:
    • Los clientes y servidores están unidos por una red.
    • El cliente puede ser una aplicación u ordenador. Demandante del servicio.
    • El servidor puede ser una aplicación u ordenador. El que ofrece el servicio.
    • Un protocolo para la comunicación entre ambos.
    • Un servicio.
    • Una base de datos, que recoge toda la información que va a requerir el servicio.
  • Tipos de arquitecturas:
    • De dos capas.
    • De tres capas.
    • De N capas.
  • Ventajas y desventajas.
  • Ejemplos de modelos cliente/servidor.

Descripción textual alternativa del video VNC

En este video se describe como se descarga el software VNC conect de la página web realvnc. Tras la descarga del software se instala y se configura en una máquina Windows. Esta máquina Windows es la máquina a cuyo escritorio queremos acceder desde un teléfono móvil. También se instala y configura el software cliente en el smartphone. Finalmente se observa como se accede desde el smartphone al escritorio de la máquina Windows.

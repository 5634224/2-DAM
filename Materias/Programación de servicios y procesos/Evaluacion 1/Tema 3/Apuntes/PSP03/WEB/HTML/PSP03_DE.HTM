Se presentan una serie de diapositivas sobre redes de comunicaciones de datos, protocolos de comunicaciones, modelo de referencia OSI, etc. Se desarrollaron redes de comunicaciones diferentes, según el tipo de señal .

Conceptos:

Transmisión de datos : Envío y recepción de señales procedentes de ordenadores.

Red de datos: Medios que permiten comunicar los ordenadores y equipos electrónicos entre sí.

Red telefónica: Conmutación de circuitos.

El mensaje se divide en paquetes (1500 bytes en Ethernet) y se les añaden cabeceras.

Cada paquete utiliza el camino más adecuado, los paquetes viajan por rutas diferentes. Al llegar al destino se ordenan.

Con la conmutación de Paquetes no se ocupan recursos. El tráfico se reparte dinámicamente. Si falla un tramo: los paquetes se envían por otro camino.

El modelo de referencia OSI define: Medio de Transmisión, características eléctricas y niveles de voltaje.

Además, se distingue entre redes de área local y redes de área extensa.

Para interconexión de redes se distingue entre conmutadores y enrutadores.

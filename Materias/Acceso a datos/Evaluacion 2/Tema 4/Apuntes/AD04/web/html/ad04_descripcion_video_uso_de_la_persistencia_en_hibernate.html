En este video tutorial se conocerán conceptos de Hibernate. y porque su uso en la lógica de negocios.

Se muestra por pantalla un código en Java con JDBC y a continuación el equivalente haciéndolo con Hibernate. También se esquematiza una aplicación con su vista, su controlador y su modelo o negocio y con ese esquema se comenta que para usar una base de datos distinta en una aplicación, por ejemplo MySQL y otra Oracle, la idea sería que podría haber algún problema con un tipo de datos concreto. Con Hibernate se consigue no tener ese problema, se hace independiente del motor de base de datos.

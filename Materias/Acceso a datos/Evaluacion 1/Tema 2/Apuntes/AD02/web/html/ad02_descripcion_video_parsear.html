Se visualiza un vídeo donde se explica cómo procesar un fichero XML mediante SAX. Se comenta que es recomendable tener conocimientos en Java XML y DTD. Se muestran trozos de código en el entorno de programación, para conseguir este fin. La idea es crear un proveedor de intérpretes, y mediante ese proveedor crear un nuevo objeto al que se le pasa el nombre del fichero xml a analizar, con el método parse. De ese modo, con muy pocas líneas se consigue analizar un fichero con SAX.

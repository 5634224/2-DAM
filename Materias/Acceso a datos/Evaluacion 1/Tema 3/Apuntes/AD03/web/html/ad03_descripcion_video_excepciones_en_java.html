Este vídeo presenta los dos mecanismos básicos que podemos emplear para la captura, propagación y gestión de excepciones en Java: utilizando la estructura try-catch es el primer mecanismo, intentado capturar la excepción que se pueda producir. Se muestra un código de aplicación en un entorno de programación, para provocar una excepción y ver el mensaje que la máquina virtual de Java produce. El otro mecanismo es declarar que un método puede provocar una excepción, mediante throw, pero no tratarla en él, sino dejar ese tratamiento para que lo trate el bloque desde donde se llamó a ese método.
